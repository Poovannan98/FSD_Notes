### Types of color format

1. name
2. RGB - Red Green Blue
3. HSL - Hue Saturation Lightness - preffered
   1. 0 < Hue < 360 deg
   2. 0 < Saturation < 100% - pop
   3. (black) 0 < Lightness < 100% (white)
4. HEX - Hexadecimal - old
   1. usual - 6 letters
   2. alpha - 8 letters
5. HWBA - Hue White Blackness Alpha - trending
   1. Hue
   2. White - x-axis
   3. Blackness - y-axis
